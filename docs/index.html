Millenaire to NBT structure converter

Blocklist

Include the base millenaire blocklist.txt here

If converting buildings from mods that use their own blocklist.txt, include those too.

Structures

Upload a zip file containing both the .txt and .png files of the building(s) you wish to convert. Use the standard Millenaire naming convention for the files.

Nested folders are supported, however if you upload too many at once be prepared for your browser to start using a lot of memory while it performs the conversion.



See The github README for known issues and limitations
